Icons

BootMetro comes with hundreds of web icons set, all available in font formats for easy coloring, sizing, and placement.

Important! Please remember to edit the file bootmetro-icons.css or bootmetro-icons.less to change the icomoon font files path according to your project structure.

Char codes are deprecrated! Use of character code like &#xe03e; in the content is DEPRECATED and is recommended to replace any use of that codes with the equivalent css class listed below.
